Smart Batch Reprocessing

Motor de procesamiento masivo asíncrono diseñado para gestionar operaciones críticas con alta tolerancia a fallos. El sistema automatiza el ciclo de vida de registros pendientes, aplicando reintentos inteligentes y garantizando la integridad de los datos mediante patrones de diseño empresariales. El objetivo fue construir un sistema capaz de procesar grandes volúmenes de datos sin saturar la memoria, manejando errores temporales (como caídas de servicios externos) y asegurando que ninguna operación se procese dos veces por error.
